## Conversor

### 📂 Descrição

Número Oculto é um jogo interativo projetado para testar suas habilidades de adivinhação. Clique no botão "Iniciar" e desafie-se a adivinhar o número oculto gerado aleatoriamente. Receba dicas úteis e veja quantas tentativas você precisa para acertar!

### 🛠️ Funcionalidades

- Adivinhação do Número Oculto: Tente adivinhar o número gerado aleatoriamente.
- Dicas Interativas: Receba dicas sobre se o número é maior ou menor que seu chute atual.
- Limite de Tentativas: Desafie-se com um limite de tentativas para acertar o número.


### 📃 Como Usar

1. Abra o arquivo `index.html` no seu navegador.
2. Clique no botão "Iniciar" para começar o jogo.
3. Insira um número para adivinhar e siga as dicas para encontrar o número oculto.
